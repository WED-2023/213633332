History and Life

She was born in Russia 1944 in city Bisk when she was 1 year old her parents return ot Polish, and she satyed 5 years in polin.
She did "Aliya" in 1950. She came on a boat in an italian boat, and she came to Maabarat "Shaa`r Haa`liya".
After 2 week in the "Maa`bara" she came to Ramle city.
She started first grade and she learned in a religous school that was called "The Mizrahi Worker" from zionist idiology
In the meantime she lived in a house with 3 other families and her father brought from polin a bag of sugar and sausages,
They sold it all for 200 pounds and bought a house without electricity and 1 room for a family of 5 people
They lived there for 2 years.
Later on, some from uncle from America came and helped her to buy a house with an electricity in year 1952
After that she got married and lived happily every after.

Q&A

Question 1

Web 1.0 is the first version of a website in the World Wide Web evolution.
The majority of the users were consumers of content.
Most of the pages were static, mean you can't change them, only rea information from them.
There were a lot of personal web pages.
They were hosted on ISP-run web servers, or free web hosting services.

Web 2.0 is the stage when websites started to highlight user generated content.
In this stage entered into the world many social media platforms like Facebook.
In contrast to Web 1.0 pages which were static, Web 2.0 pages were interactive.
Users can click on things and change how the website looks. Therefore, people could put like and share and those
applications made social media websites works.
We can say that Web 2.0 is a better version of Web 1.0.

Web 3.0 is the stage when website start using Artifical Intelligence and Machine Learning
In contrast to Web 2.0 which has been mainly about front end user experience, Web 3.0 Focuses more about the back end.
In Web 3.0 the Web was start to acting like a huge database.
Web 3.0 also features the Semantic Web.
The semantic web means an improve way of searching in the web, what lead to better results from search engines.
In the Semantic Web we could make an analysis based on the capability to comprehend the meaning of words, rather than on keywords or numbers.
For example, saving words as vectors instead as words themselves in order to make words with similar meaning a close distance between each vector

Question 2

One difference between web 1, web 3 can be explained through a use case :
For example, imagine we search for "side effect of a new medicine"
If we search this in web 1, we would get a static page about many other drugs, and we would have to extract from there by ourselves information about the specific medicine we want.
In web 3, thourgh use of a the semantic web, we could get a page that can give us better results for the medicine that we looked for, and this could be a page dedicated only to this specific medicine.
we might even get a page of the company that produced this medicine and we could extract the information from There and not from a random page in wikipedia.
This is because through semantic web we can find similar words in meaning to the medicine we looked for, because of the words' vectors that this web represents.

Another difference we could see between web 1 and web 2 if we try to open a page for some social media, lets take facebook for example.
If I would have opened a page of facebook in web 1 I would get a static page of a user, without the ability to comment, share, or like/dislike any image or video that I would see there.
In contrast, in web 2 the page would be dynamic, and I could do all the previous actions I have mentioned before and it would be much more joyful and addictive.
